# DataForgeStudio V4 快速开始指南

## 系统要求

### 浏览器要求
- Chrome 90+ (推荐)
- Firefox 88+
- Edge 90+
- Safari 14+

### 分辨率要求
- 最低分辨率: 1280x1024
- 推荐分辨率: 1920x1080 或更高

### 网络要求
- 与服务器正常连接
- 建议使用有线网络或稳定的 WiFi

---

## 首次登录

### 步骤 1: 访问系统

在浏览器地址栏输入系统地址，例如：
```
http://your-server:port
```

### 步骤 2: 输入凭据

系统默认管理员用户：
- **用户名**: `root`
- **密码**: 查看服务器控制台输出的临时密码

> **重要**: root 用户是系统内置用户，拥有所有权限，不可被删除。

### 步骤 3: 修改密码

首次登录后，系统会要求您修改密码：

1. 输入临时密码
2. 输入新密码（至少 6 位）
3. 确认新密码
4. 点击"确定"完成修改

> **安全提示**: 请使用强密码，包含大小写字母、数字和特殊字符。

## 基本操作

### 导航菜单

系统左侧为导航菜单，包含以下模块：

| 模块 | 说明 |
|------|------|
| 首页 | 系统概览和快捷入口 |
| 报表管理 | 报表设计、查询和导出 |
| 许可管理 | 许可证激活和状态查看 |
| 系统管理 | 用户、角色、数据源、日志等 |

### 创建报表

1. 点击左侧菜单 **报表管理** → **报表设计**
2. 点击右上角 **新建报表** 按钮
3. 填写报表基本信息：
   - 报表名称
   - 报表分类
   - 选择数据源
4. 编写 SQL 查询语句
5. 点击 **测试查询** 验证 SQL
6. 配置字段显示和参数
7. 点击 **保存** 完成创建

### 查看报表

1. 点击左侧菜单 **报表管理** → **报表查询**
2. 选择要查看的报表
3. 输入查询条件（如果有参数）
4. 点击 **查询** 按钮
5. 查看、导出数据

## 常见问题

### Q: 忘记密码怎么办？

**A**: 联系系统管理员通过以下方式重置：
1. 登录系统管理后台
2. 进入 **系统管理** → **用户管理**
3. 找到对应的用户，点击 **重置密码**
4. 将新密码告知用户

### Q: 报表查询速度慢怎么办？

**A**: 可以尝试以下方法：
- 添加 WHERE 条件限制数据量
- 优化 SQL 查询语句
- 为数据库表添加索引
- 联系管理员进行数据库优化

### Q: 无法导出 Excel？

**A**: 检查以下事项：
1. 确认是否有"导出 Excel"权限
2. 检查数据量是否过大（超过 10 万行可能需要较长时间）
3. 检查浏览器是否允许下载文件
4. 刷新页面后重试

### Q: SQL 查询被拦截？

**A**: 系统的 SQL 安全机制会拦截危险语句：
- 只允许 SELECT 查询
- 不允许 DROP、DELETE、INSERT、UPDATE 等 DML/DDL 语句
- 不允许注释字符（-- 和 /* */）
- 不允许 UNION 注入
- 不允许分号（多语句）

请确保您的 SQL 查询符合安全规范。

### Q: 如何备份数据？

**A**: 联系系统管理员：
1. 进入 **系统管理** → **备份管理**
2. 点击 **创建备份**
3. 等待备份完成
4. 下载备份文件到本地保存
