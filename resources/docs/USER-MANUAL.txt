【1. 系统概述】

DataForgeStudio 是一款专业的数据报表管理系统，支持多数据源查询和动态报表设计。
本手册将帮助您快速了解和使用系统的各项功能。


【2. 功能模块介绍】

2.1 报表查询
-----------
- 执行已设计的 SQL 报表
- 支持参数化查询条件
- 数据导出为 Excel 或 CSV 格式
- 查看查询历史记录

2.2 报表设计
-----------
- 可视化 SQL 报表编辑器
- 支持参数定义和条件筛选
- 语法高亮和自动补全
- 报表预览和测试

2.3 数据源管理
-------------
- 支持多种数据库：SQL Server、MySQL、PostgreSQL、Oracle、SQLite
- 连接测试功能
- 密码加密存储
- 数据库列表查看

2.4 用户管理
-----------
- 用户账号创建与编辑
- 角色权限分配
- 用户状态管理

2.5 角色管理
-----------
- 角色创建与配置
- 功能权限设置
- 用户关联管理

2.6 许可管理
-----------
- 查看当前许可证状态
- 机器码查看
- 许可证导入

2.7 日志管理
-----------
- 操作日志查询
- 按时间、用户、操作类型筛选
- 日志导出

2.8 备份管理
-----------
- 数据库备份
- 备份文件管理
- 数据恢复


【3. 快速入门】

步骤 1：登录系统
   使用管理员账号登录系统

步骤 2：配置数据源
   进入【数据源管理】，点击【新增】添加数据库连接
   填写服务器地址、数据库名、用户名密码
   点击【测试连接】验证配置

步骤 3：创建报表
   进入【报表设计】，点击【新增】创建报表
   编写 SQL 查询语句，定义查询参数
   保存并测试报表

步骤 4：执行报表
   进入【报表查询】，选择报表
   设置查询参数（如有），点击【查询】
   查看结果并可导出数据


【4. 常见问题】

Q: 忘记密码怎么办？
A: 请联系系统管理员重置密码。

Q: 报表查询超时怎么办？
A: 优化 SQL 查询语句，或联系管理员调整超时设置。

Q: 如何更换许可证？
A: 进入【许可管理】，导入新的许可证文件即可。

Q: 支持哪些数据库？
A: 支持 SQL Server、MySQL、PostgreSQL、Oracle、SQLite。
