【产品简介】

DataForgeStudio 是一款专业的数据报表管理系统，提供动态 SQL 报表设计、
多数据源支持、权限管理等功能，帮助企业高效管理和分析数据。


【主要功能】

1. 报表管理
   - 动态 SQL 报表设计
   - 参数化查询
   - 数据导出（Excel、CSV）

2. 数据源管理
   - 支持 SQL Server、MySQL、PostgreSQL、Oracle、SQLite
   - 连接测试
   - 密码加密存储

3. 用户权限
   - 用户管理
   - 角色权限控制
   - 操作日志记录

4. 系统管理
   - 数据备份与恢复
   - 许可证管理
   - 系统配置


【快速入门】

1. 登录系统（默认管理员账号：root）
2. 配置数据源（系统管理 -> 数据源管理）
3. 设计报表（报表设计 -> 新建报表）
4. 查询数据（报表查询 -> 选择报表）
