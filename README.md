# DataForgeStudio V1.0.0

## 产品简介

DataForgeStudio 是一款面向企业的智能报表设计与数据集成平台，帮助企业快速构建、管理和发布动态报表。

### 核心功能

- **动态SQL报表设计器** - 可视化界面，支持参数化查询
- **多数据源支持** - SQL Server、MySQL、PostgreSQL、Oracle等
- **报表查询与导出** - Excel、CSV格式导出
- **用户权限管理** - 细粒度的角色权限控制
- **许可证管理** - 机器码绑定，15天免费试用

### 技术栈

- 后端：ASP.NET Core 8.0, Entity Framework Core
- 前端：Vue 3, Element Plus
- 数据库：SQL Server 2005+

### 快速开始

1. 下载安装包 `DataForgeStudio-Setup.exe`
2. 运行安装程序
3. 配置数据库连接
4. 启动服务，访问 http://localhost:8089

### 默认账户

首次运行时，系统会自动生成 root 用户的临时密码，请查看控制台输出。
